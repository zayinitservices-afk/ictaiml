# Public Files Directory

Place your downloadable files here. Files in this directory are served statically and accessible via URLs.

## Required Files for CFP Page

Please add the following files to this directory:

1. **template.docx** - Microsoft Word template for paper submissions
2. **template.zip** - LaTeX template (zip file containing .tex and related files)
3. **author-checklist.pdf** - Author submission checklist (optional but recommended)

## File Structure

```
public/
  ├── template.docx          (Word template)
  ├── template.zip           (LaTeX template archive)
  └── author-checklist.pdf   (Author checklist)
```

## How to Add Files

1. Copy your template files into this `public` folder
2. Make sure filenames match exactly:
   - `template.docx` (for Word template)
   - `template.zip` (for LaTeX template - can contain .tex, .sty, .cls files)
   - `author-checklist.pdf` (optional)

3. Files will be accessible at:
   - `/template.docx`
   - `/template.zip`
   - `/author-checklist.pdf`

## Notes

- File names are case-sensitive
- Maximum file size recommended: 10MB per file
- These files will be publicly accessible, so only include files you want users to download
